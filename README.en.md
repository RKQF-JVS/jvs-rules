# jvs-rules（规则引擎 决策引擎）

#### Description
本项目是基于JVS逻辑引擎构建的规则引擎，主要用于风控决策、规则过滤、行为评分等场景，本项目采用spring cloud+vue构建，支持在线的变量加工、界面拖拽、在线测试等多种功能。包含完整的前后端。
